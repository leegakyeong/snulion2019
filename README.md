# Django study for snulion7th
## django cheat sheet
새 프로젝트 만들기
```bash
$ django-admin startproject myproject
$ cd myproject
```
새 어플리케이션 만들기
```bash
$ django-admin startapp myapp
```
서버 켜기
```bash
$ python manage.py runserver
```
migrate
```bash
$ python manage.py makemigrations
$ python manage.py migrate
```
장고 쉘
```bash
$ python manage.py shell
```
